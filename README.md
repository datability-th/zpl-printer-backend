# zpl-printer-backend

## How to run

```bash
node app.js
```

## Installation

### Dependency

```bash
"node": v15.11.0
"npm": 7.6.0
"yarn": 1.22.5
```

Note: If you have problem about `Printer Package`

```
npm install printer --target_arch=ia32
npm install printer --target_arch=x64
```

more detail - https://www.npmjs.com/package/printer
